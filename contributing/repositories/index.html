On Github

To ease making patches, the common developpement branches of the OAR repositories are pushed on Github. You'll find the repository here :

https://github.com/oar-team/oar

read-only access

git clone git://github.com/oar-team/oar.git

developper access

git clone git@github.com:oar-team/oar.git

On the Inria GForge

The OAR project is hosted by the INRIA Gforge at https://gforge.inria.fr/projects/oar/.

  • Source repository:

    anonymous access

    git clone git://scm.gforge.inria.fr/oar/oar.git

    developper access

    git clone git+ssh://developername@scm.gforge.inria.fr//gitroot//oar/oar.git
