There is currently 2 main branches for OAR:

  1. For the 2.5.x series
    1. Debian
    2. RPM
    3. Source Tarball
  2. For the 2.4.x series
    1. Debian
    2. RPM
    3. Source Tarball
  3. Projects around oar
    1. oargrid

For the 2.5.x series

This branch is considered as an stable branch.

Debian

You need to install first the keyring correponding the this repository:

  • by installing the package oar-keyring
  • by executing:
wget -q http://oar-ftp.imag.fr/oar/oarmaster.asc -O- | sudo apt-key add -

You need to choose one of the branch according to your needs

OAR stable releases

deb http://oar-ftp.imag.fr/oar/2.5/debian/ sid main
deb http://oar-ftp.imag.fr/oar/2.5/debian/ squeeze main

OAR testing releases

deb http://oar-ftp.imag.fr/oar/2.5/debian/ sid-proposed-updates main
deb http://oar-ftp.imag.fr/oar/2.5/debian/ squeeze-proposed-updates main

OAR snapshot builds

deb http://oar-ftp.imag.fr/oar/2.5/debian/ sid-snapshots main
deb http://oar-ftp.imag.fr/oar/2.5/debian/ squeeze-snapshots main

RPM

The rpm packages of the 2.5 branch are currently targeted to CentOS 6

http://oar-ftp.imag.fr/oar/2.5/rpm/stable
http://oar-ftp.imag.fr/oar/2.5/rpm/testing
http://oar-ftp.imag.fr/oar/2.5/rpm/unstable

Source Tarball

http://oar-ftp.imag.fr/oar/2.5/sources/stable
http://oar-ftp.imag.fr/oar/2.5/sources/testing
http://oar-ftp.imag.fr/oar/2.5/sources/snapshots
